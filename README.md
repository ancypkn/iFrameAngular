# IframeAngularApp

A simple Angular app that demonstrates using iframes to load a child app in Parent angular Application.

## Installation

1. Clone the repository.
2. Run `npm install`.
3. run the child Application at 'http://localhost:4201/'  - ng serve --port=4201 where you see the iframe embeded
  run the parent Application at 'http://localhost:4200/  - ng serve
